# Интерактивная панель управления 

Интерактивная панель управления для мониторинга лабораторного оборудования. Проект выполнен в рамках обучения/тестового задания (укажи свое).

## Демо проекта
Посмотреть проект вживую можно здесь: [https://vlyd14ka.github.io/Biocad/](https://vlyd14ka.github.io/Biocad/)



##  Структура проекта
- `/card` — логика, стили и шаблоны карточек.
- `/images` — графические ресурсы (SVG/PNG иконки).
- `/error` — логика обработки ошибок.
- `index.html` — главная страница дашборда.
- `analytics.html` — детальная информация об устройстве.
